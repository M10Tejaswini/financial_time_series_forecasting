# Stock Price Forecasting using LSTM

This project predicts stock prices (AAPL example) using an LSTM neural network.

## Structure
- `src/` : Python scripts for data, model, prediction, visualization
- `plots/` : Predicted vs Actual charts
- `data/` : (Optional) store historical CSVs
- `run_forecast.py` : Main script to run the project
- `requirements.txt` : Python dependencies

## Usage
1. Install dependencies: `pip install -r requirements.txt`
2. Run the main script: `python run_forecast.py`
3. Check generated plots in `plots/` folder.
